# React-and-FastApi-To-Do-
A small website called To-Do is built using React and FastApi for educational purposes.
To launch the site, you need to:
Run React with these commands
cd To-Do
npm run dev
Run FastApi with these commands
cd To-Do
uvicorn backend.main:app --reload --port 8000 (or another port if port 8000 is busy)
